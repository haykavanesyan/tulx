# Documentation Example

This is an example MDX page demonstrating how to write documentation.

## Features

MDX allows you to:

- Write **markdown** with full support for JSX
- Use React components directly in your documentation
- Create interactive documentation

## Code Examples

Here's an example of inline code: `const example = 'Hello World'`

And here's a code block:

```typescript
function greet(name: string) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
